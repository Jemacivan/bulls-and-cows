A logical game written in python - Bulls and Cows

The computer generates a random 4 digit number that you have to guess. If a digit in your number is right but not in its place, it counts as a cow, if it is right and in its place, it counts as a bull. Gather 4 bulls for a win.

git clone https://github.com/Jemacivan/bulls-and-cows

cd bulls-and-cows

python main.py
